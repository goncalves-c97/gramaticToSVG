A gramática utilizada foi: "E+E+E+E+EE--E+EEE+E", porém pode ser alterada normalmente no arquivo "gramatica.txt".

E a única regra definida foi a seguinte: "E-EE-EEE-EE-E+EE++E+++EEE--EE--E".

Onde cada 'E' refere-se a andar, o '+' refere-se a virar para direita e '-' virar para esquerda, assim como no exemplo mostrado no livro durante a aula.
